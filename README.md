
## Projeto 'Post its' - Semana de React Js da {reprograma}

Esse projeto foi desenvolvido com o [Create React App](https://github.com/facebook/create-react-app).

### Funcionalidades:
* Página de Login e Registro
* Salva o usuário para logar na página principal
* Página Home com o espaço para criar post its
* Mudar a cor dos post its
* Deletar os posts its 
* Editar os posts its
* Sair da página e automaticamente deslogar o seu usuário

Esse projeto aceita pull requests :panda_face 

---

### `npm start`

Rode esse aplicativo localmente.<br>
A porta [http://localhost:3000](http://localhost:3000) é aberta para visualização no browser

A página irá atualizar automaticamente se você fizer edições .<br>
Ele também indica no console se algum erro acontecer.
